3. WCAG textOnPrimary
Fórmula luminosidad relativa: L = 0.2126*R + 0.7152*G + 0.0722*B
Si L > 0.179 → texto negro; si no → texto blanco
4. CampaignConfig parsea sección sponsor
Cuando cambia la campaña → SponsorAssets se actualiza → todos los componentes UI reciben el nuevo branding
